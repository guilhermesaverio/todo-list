 📝 To-Do List em Java

Projeto simples de To-Do List feito em Java, com menu de console.

- **model.Tarefa** → representa uma tarefa com descrição e status de conclusão.  
- **service.TodoService** → gerencia a lista de tarefas (adicionar, listar, concluir, remover).  
- **TodoList** → classe principal com menu de console para interação com o usuário.

O projeto demonstra conceitos de **POO, listas e controle de fluxo**, com estrutura limpa e modular.
